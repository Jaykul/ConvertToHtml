# ConvertToHtml
PowerShell Module to convert PSObjects to formatted Html

Plan to publish to PowerShell Gallery, but currently to install, 
Clone the repo,
Copy the ConvertToHtml folder to C:\Program Files\WindowsPowerShell\Modules
